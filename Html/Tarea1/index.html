Etiquetas de titulo

En HTML tenemos 6 niveles de títulos comprendidos como las etiquetas < h1>, < h2 >, < h3 >, < h4 >, < h5 > y < h6 >, cuya función principal es la de poder jerarquizar los bloques de contenido

Etiquetas de tablas

Dentro del lenguaje de diseño web, las tablas HTML se crean usando las etiquetas < table > y < /table >. En ella se incluyen dos etiquetas importantes: < tr >, que es para crear filas de tablas y < td >, para crear celdas de datos. Todo lo que esté dentro de ambas etiquetas es el contenido de la celda de la tabla.

Etiquetas de parrafo

El elemento p (párrafo) es el apropiado para distribuir el texto en párrafos. Sus etiquetas son: < p> y < /p> (la de cierre es opcional). Crea una caja: en bloque. Está definido como: Elemento de frase.

Etiquetas multimedia

< image> Es la más común para insertar imágenes. Como las otras etiquetas para insertar elementos multimedia en HTML, tiene un atributo src que determina la ubicación del archivo que queremos insertar.

< picture> Representa un contenedor que provee múltiples fuentes o atributos < source> al elemento img contenido dentro de él. Es decir, dentro del contenedor puede haber distintas versiones de una imagen o incluso distintas imágenes.

< figure> La diferencia entre la etiqueta < figure> y la etiqueta < img> se basa en el elemento < figcaption>.

Etiquetas de listas

Etiquetas < li>,< ul>, < ol> Las listas se usan para citar, numerar y definir cosas. Cada uno de los elementos de la lista comenzará con una etiqueta < li>, que puede o no llevar su correspondiente < /li> de cierre.
